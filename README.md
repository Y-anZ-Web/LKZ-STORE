# LKZ Shop 🛒

Sua loja digital de produtos gamers.

## Sobre

A LKZ Shop oferece ferramentas, launchers e recursos digitais premium para a comunidade gamer.

## Funcionalidades

- Produtos digitais com carrinho de compras
- Sistema de login/cadastro (localStorage)
- Modo claro/escuro
- Console secreto (tecla `~`)
- Design responsivo
- Carrinho lateral com cálculo de total

## Produtos

- Lua Tools
- Sky Tools
- Hydra Launcher
- Nuvio Cloud
- Optimizer Pro
- Ping Reducer
- Texture Pack 4K
- Mod Pack Elite
- AntiCheat Pro

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
- localStorage

## Como usar

1. Clone o repositório
2. Abra o `index.html` no navegador
3. Ou acesse via GitHub Pages
